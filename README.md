# Vroozi Demo

A demo project by [Shapestone](https://shapestone.com) showcasing integration with [Vroozi](https://vroozi.com), a modern procurement and spend management platform.

## Overview

This repository contains demo code and examples for working with the Vroozi platform, including procurement workflows, purchase order management, and spend analytics integrations.

## Getting Started

Clone the repository:

```bash
git clone git@github.com:shapestone/vroozi-demo.git
cd vroozi-demo
```
